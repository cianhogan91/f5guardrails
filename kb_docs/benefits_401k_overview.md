# 401(k) — Quick Internal Explainer (Approved)

## What a 401(k) is
A 401(k) is an employer-sponsored retirement plan that allows employees to contribute from payroll.

## Key characteristics
- Contributions can be pre-tax or Roth (depending on plan options).
- Contributions typically come directly from payroll deductions.
- Some employers offer matching contributions.
